# Agent Instructions

## Scope
This is a Neovim configuration repository. Stay inside this project folder. Do not read, write, or execute anything outside of it.

## Communication
- No unsolicited explanation. Code is self-documenting.
- Only explain when explicitly asked.
- No summaries of what you just did unless asked.
- No "here's what I changed" paragraphs. Let the diff speak.

## Research & Web Fetches
All web fetches, URL lookups, and external documentation research must be delegated to subagents. Do not fetch URLs inline in the main agent.

## Code Style
Follow the patterns already established in the repo. Read before writing.
